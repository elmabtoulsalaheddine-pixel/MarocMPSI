## 📘 Série 1 : Les nombres réels

### 🎯 Objectifs
Cette série d’exercices a pour but de renforcer les notions fondamentales sur les nombres réels, notamment :

- Les intervalles de $\mathbb{R}$  
- Les majorants et minorants  
- La borne supérieure et la borne inférieure  
- La densité de $\mathbb{Q}$ dans $\mathbb{R}$

---

### 📌 Conseils
- Lire attentivement chaque énoncé avant de commencer  
- Rédiger proprement les démonstrations  
- Vérifier chaque étape du raisonnement  
- Éviter les calculs inutiles
